# Git시작
  - 1. git-scm.com에 가서 git을 다운받는다.(모두 기본 값으로 다운)
  - 2. git bash응용프로그램을 실행 시킨다
  - 3. 원하는 폴더에 가서 "git init" 명령어를 실행한다
    - .git파일이 만들어진다(버전관리에 관한 정보들 담김)
