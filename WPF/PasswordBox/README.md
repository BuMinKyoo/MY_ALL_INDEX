
##### - PasswordBox : TextBox와는 다르게, 비밀번호처럼 감춰지는 형태로 사용할 수 있다.
  - 'Text'속성 대신 'Password'속성을 이용하여 글자 입력
  - PasswordChar속성을 이용해서 비밀번호가 감춰질때 의 도형 설정 가능

***

<br/>

#XAML
~~~
<Grid>
    <PasswordBox Width="100" Password="비밀번호" VerticalAlignment="Center"/>
</Grid>
~~~
