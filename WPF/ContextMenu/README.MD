
##### - ContextMenu : 마우스 오른쪽 클릭 했을때 나오는 Menu창

<br/>

~~~
<Grid>
    <TextBox>
        <TextBox.ContextMenu>
            <ContextMenu>
                <MenuItem Header="전체 지우기"/>
                <MenuItem Header="전체 선택" InputGestureText="Ctrl+A"/>
            </ContextMenu>
        </TextBox.ContextMenu>
    </TextBox>
</Grid>
~~~
